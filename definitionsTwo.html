Definitions

Group Two

  1. Sequential: Executes instructions in a linear order, one after another, following the program flow.
  2. Conditional: Enables decision-making based on conditions, executing different code paths depending on whether a condition is true or false.
  3. Iterative (Looping): Repeats a set of instructions iteratively until a specific condition is met, allowing efficient handling of repetitive tasks.

Hypertext Markup Language (HTML) is the standard language used to create and structure content on the World Wide Web. It uses markup tags to define elements like headings, paragraphs, images, links, and more. Markup languages are sets of tags and rules that define the structure and presentation of documents, facilitating content display on web browsers.

Cascading Style Sheets (CSS) is a style sheet language used in web development to control the presentation and layout of HTML documents. CSS defines how elements are displayed on web pages, including fonts, colors, margins, padding, and positioning, allowing web designers to separate content from its visual representation for consistent and appealing layouts.

Programming language syntax refers to the set of rules governing the structure and arrangement of code elements. It defines how instructions are written to convey commands to a computer. Programming languages are used to create software applications, websites, and other computational tasks, enabling developers to communicate with computers and achieve specific functionalities.
